# Silvia Matos - Estética Avançada (ERP & CRM) 🥂🚀

Sistema de gestão completo desenvolvido para o setor de estética, focado em experiência do usuário (UX), controle financeiro e fidelização de clientes.

## 🛠️ Tecnologias Utilizadas
- **Framework:** Next.js 14 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS (Design Black & Rose Gold)
- **Banco de Dados & Auth:** Supabase (PostgreSQL)
- **Ícones:** Lucide-React

## 💎 Diferenciais do Projeto
- **Arquitetura Escalável:** Uso de Hooks customizados e componentes reutilizáveis.
- **Segurança de Dados:** Middleware de autenticação que protege rotas sensíveis e garante que apenas usuários autorizados acessem os dados.
- **Foco em Business Intelligence:** - Cálculo automático de status VIP baseado em volume de compras.
  - Alerta de aniversariantes do mês para ações de marketing.
  - Histórico de consumo detalhado integrado ao prontuário da cliente.
- **Interface Mobile First:** Desenvolvido especificamente para ser utilizado em telemóveis, garantindo que a Silvia possa gerir o negócio de qualquer lugar.

## 🚀 Como Executar
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure as variáveis de ambiente `.env` com suas chaves do Supabase.
4. Rode o projeto: `npm run dev`.
